# 💰 Telegram Бот-Счетовод для компаний

Бот помогает делить совместные расходы в группах и автоматически рассчитывает, кто кому и сколько должен. Суммы распределяются как на всех участников, так и на конкретных людей.

## 🛠️ Технологии
* Python 3.11+
* Aiogram 3.x (Фреймворк для бота)
* Aiosqlite (Асинхронная база данных SQLite)

## 🚀 Запуск проекта
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`
3. Задайте переменные окружения:
   * `BOT_TOKEN` — токен вашего бота от BotFather.
   * `ADMIN_ID` — ваш Telegram ID.
4. Запустите файл бота: `python bot.py`
