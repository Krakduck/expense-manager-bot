# 🤖 MultiBuddy Bot — Многофункциональный помощник для Telegram-групп

**MultiBuddy Bot** — это асинхронный Telegram-бот, разработанный для упрощения взаимодействия в групповых чатах. Бот совмещает в себе финансовый трекер расходов, систему отслеживания регулярных событий (habit tracker), базу цитат участников и кастомные автоответы на упоминания.

---

## ✨ Ключевые возможности

### 💰 Учет совместных расходов и долгов
* **Гибкое распределение:** Возможность записать трату на всех участников чата или на конкретного человека (`/spent`).
* **Оптимизация взаиморасчетов:** Алгоритм автоматически рассчитывает финальный баланс группы и строит кратчайшую цепочку переводов для закрытия долгов (`/debts`).
* **Администрирование:** Быстрый сброс истории расходов группы модератором (`/clear_debt`).

### 📅 Трекер регулярных событий и серий (Habit Tracker)
* **Контроль привычек:** Создание групповых или личных целей с автоматическим или ручным продлением серии дней (`/event`).
* **Фоновые задачи:** Интеграция с планировщиком задач для ежедневной проверки и обновления состояний в режиме реального времени.

### 💬 Интерактивный цитатник и утилиты
* **Сохранение цитат:** Возможность сохранить любое важное или смешное сообщение в базу данных в один клик через Reply (`/save`).
* **Умный поиск:** Просмотр общего архива фраз или фильтрация цитат по конкретному автору (`/citations`).
* **Автоответы на пинг:** Пользователи могут задать фразу в профиле (`/profile`), которой бот будет автоматически отвечать, если их тегнут в чате.
* **Инструменты вовлечения:** Команды для быстрого призыва всех участников (`/all`) и выбора случайного пользователя (`/random`).

## 🛠️ Технологический стек

* **Язык разработки:** Python 3.11+
* **Фреймворк:** [Aiogram 3.x](https://github.com/aiogram/aiogram) (Асинхронная работа с Telegram Bot API)
* **База данных:** [Aiosqlite](https://github.com/nolar/aiosqlite) (Асинхронная обертка для SQLite3 с оптимизированными индексами для быстрого поиска)
* **Планировщик задач:** [APScheduler](https://github.com/agronholm/apscheduler) (AsyncIOScheduler для управления временными триггерами событий)
* **Архитектура:** Конечные автоматы (FSM) для пошаговых сценариев, изолированные роутеры, HTML-экранирование для защиты от ошибок разметки Telegram.
